## Getting Started

### Prerequisites

- GoLang 1.19
- A MongoDB database
- Basic understanding of Go and MongoDB

### Installing

0. ```go install github.com/swaggo/swag/cmd/swag@latest```
2. Create a .env file
3. ```go run main.go```
4. Visit http://localhost:8080/docs

### Scripts

- ```swag init``` - Generates the swagger docs
